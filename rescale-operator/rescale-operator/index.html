AI Automation OS for Creative & Campaign Operations

Problem

Creative and campaign operations were fragmented across multiple tools, dashboards, naming conventions, creative assets, and team workflows. Teams had to manually review campaign setup, naming errors, performance signals, launch readiness, and winner/loser decisions — creating delays, inconsistent QA, reporting errors, and missed opportunities to scale winning creatives quickly.

Solution

Built a workflow-driven AI automation system that combines campaign QA, creative signal filtering, winner detection, launch checklists, dashboards, and human-in-the-loop review. Rescale Operator gives teams one structured operating layer instead of juggling disconnected tools and manual spreadsheets.

Technical breakdown

Production architecture: data inputs, processing logic, human review, failure handling, and stack.

Data inputs
  • Campaign / ad / ad set metadata
  • Creative naming fields
  • Spend, ROAS, CTR, CPC, CPM
  • Product and market data
  • Creative status and launch tracking
  • Manual team inputs
Processing logic
  • Parse naming convention (NC v3)
  • Validate required fields
  • Classify QA status (valid / partial / invalid)
  • Apply winner / loser rules
  • Prioritize and filter signals
  • Log reviewer actions
Human review
  • Business owner reviews flagged issue
  • Accept / reject / fix recommendation
  • Action is logged with timestamp and reviewer
  • Dashboard updates status in real time
Failure handling
  • Missing data fields → flagged for manual review
  • Invalid naming format → routed to QA queue
  • API / data sync failure → retry with alerting
  • Rule conflict → manual override required
Tools and stack
  • Frontend: Next.js, React, TypeScript, TanStack Query, Tailwind
  • API: FastAPI, Pydantic, feature-flagged routes
  • Workers: Arq + Redis (async winner checks, cron fan-out)
  • Data: Supabase Postgres, Ecom Profits read-only Meta view
  • AI: Claude/Gemini/OpenAI workflows, Supermemory RAG, agent memory
  • Integrations: ClickUp, Google Sheets, Meta Ads data pipelines
  • Quality: pytest, Vitest, Playwright e2e, Docker full stack
